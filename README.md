# Weather Agent Chat Interface

##  Setup & Installation

1. **Clone the repository**:
   ```bash
   git clone[ <repo-url>](https://github.com/Aadikamath080410/Weather-agent-chat-interface-.git)
   cd weather
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```
